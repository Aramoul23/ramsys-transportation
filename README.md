# Ramsys School App 2

A comprehensive school bus routing and transportation management system.

## Features
- Dynamic route optimization for school buses.
- Student pickup and drop-off management.
- Real-time ETA calculations.
- Admin dashboard for scenario building and fleet management.

## Tech Stack
- **Backend**: Python (Flask)
- **Database**: SQLite
- **Routing Engine**: OSRM (Open Source Routing Machine)
- **Maps data**: OpenStreetMap (OSM)
- **Frontend**: HTML/CSS/JS (Vanilla)

## Setup
1. Install dependencies from `requirements.txt` (if available) or manually install Flask, etc.
2. Initialize the database using `init_db.py`.
3. Start the server using `python app.py`.
